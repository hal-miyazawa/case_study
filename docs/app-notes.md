# App Notes

Git で共有するための、このアプリに関する作業メモです。

## 概要

- プロジェクト名: `case_study`
- 種別: React + TypeScript + Vite のフロントエンドアプリ
- 現在の画面: テキストボックスをクリックするとセリフが進み、途中で選択肢を選べる1画面デモ
- 主な目的: 災害予防・防災に関するケーススタディ用アプリとして拡張する想定
- 企画詳細: [project-plan.md](project-plan.md)
- 最小版のストーリー流れ: [story-flow.md](story-flow.md)

## 技術構成

- React: `19.2.6`
- React DOM: `19.2.6`
- TypeScript: `6.0.2`
- Vite: `8.0.12`
- ESLint: `10.3.0`

## 主要ファイル

- `src/main.tsx`: React アプリのエントリーポイント
- `src/App.tsx`: 現在の主要 UI コンポーネント
- `src/App.css`: `App.tsx` 用のスタイル
- `src/index.css`: グローバルスタイル、色、フォント、レスポンシブ設定
- `src/assets/hero.png`: ヒーロー領域で使っている画像
- `public/icons.svg`: 画面内リンク用の SVG アイコンスプライト

## 実行コマンド

```bash
npm install
npm run dev
npm run build
npm run lint
```

## 現在の実装メモ

- `App.tsx` は `dialogues` 配列と `useState` で、現在表示するセリフ番号を管理している。
- 黒いテキストボックスをクリックすると、次のセリフに進む。
- 最後のセリフまで進むと、左右の大きな選択肢ボタンが表示される。
- 選択肢を押すと、選んだ内容がテキストボックスに表示される。
- 選択後にテキストボックスを押すと、最初のセリフへ戻る。
- `App.css` では、簡易的な部屋背景、主人公、テキストボックス、名前欄を CSS で表現している。
- 現段階では、別画面への遷移や本格的なシナリオ分岐はまだ入れていない。
